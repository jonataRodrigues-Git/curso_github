Comandos Git e GitHub

Comando Descrição
Comandos Básicos do Git
git init Inicializa um novo repositório Git local.
git clone <URL> Clona um repositório remoto para sua máquina local.
git status Mostra o estado atual do repositório.
git add <arquivo> Adiciona um arquivo à área de staging.
git add . Adiciona todos os arquivos modificados para staging.
git commit -m "mensagem" Cria um commit com uma mensagem.
git push Envia os commits locais para o repositório remoto.
git pull Puxa alterações do repositório remoto para o local.
git fetch Busca mudanças do remoto, sem mesclar.
git merge <branch> Mescla outra branch à atual.
Trabalhando com Branches
git branch Lista todas as branches locais.
git branch <nome> Cria uma nova branch.
git checkout <branch> Troca para outra branch.
git checkout -b <nome> Cria e troca para nova branch.
git branch -d <branch> Deleta uma branch local.
git switch <branch> Troca de branch (moderno).
Histórico e Revisões
git log Mostra o histórico de commits.
git log --oneline Histórico compacto.
git diff Mostra diferenças não commitadas.
git show <commit> Exibe detalhes de um commit.
Desfazer Mudanças
git reset Remove arquivos do staging.
git reset --hard Descarta todas modificações locais.
git checkout -- <arquivo> Restaura o arquivo para última versão commitada.
git revert <commit> Desfaz um commit específico com novo commit.
Limpeza e Correções
git clean -f Remove arquivos não rastreados.
git stash Salva modificações temporárias.
git stash pop Recupera modificações salvas com stash.
Git Remoto e GitHub
git remote -v Lista repositórios remotos.
git remote add origin <url> Adiciona repositório remoto.
git push -u origin <branch> Faz push e define branch remota padrão.
git remote remove origin Remove um repositório remoto.
Configuração do Git
git config --global user.name "Seu Nome" Define o nome de usuário.
git config --global user.email "email@exemplo.com" Define o e-mail do usuário.
git config --list Exibe configurações ativas.
GitHub (via terminal)
gh auth login Login no GitHub via terminal.
gh repo create Cria repositório GitHub.
gh repo clone <user>/<repo> Clona um repositório GitHub.
gh issue list Lista issues.
gh pr create Cria Pull Request.
gh pr list Lista Pull Requests.
Outros Comandos Úteis
git tag <versão> Marca uma versão com tag.
git blame <arquivo> Mostra quem alterou cada linha.
git archive Cria .zip ou .tar do projeto atual.
